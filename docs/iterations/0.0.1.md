# 0.0.1 — WeChat Web Chat 首版

## 范围

纯前端微信风格聊天界面，单文件 HTML/CSS/JS，假数据驱动。

### 功能清单

1. 联系人列表：左侧展示至少 3 个联系人，高亮选中态
2. 聊天区域：右侧展示当前对话消息气泡
3. 消息气泡：区分发送者（我方绿泡、对方白泡）、时间戳
4. 输入框+发送按钮：底部消息编辑区，回车或点击发送
5. 模拟数据：联系人名称/头像、聊天记录硬编码
6. 响应式布局：微信桌面版经典三栏布局

### 不做

- 后端 API、数据库、用户登录
- 文件/图片/语音/视频
- 实时消息推送
- 移动端适配

### 技术方案

- 单文件 `index.html`，内联 CSS + 原生 JS
- 无构建工具、无 npm 依赖
- 部署：Python http.server 绑定 `127.0.0.1:19002`
- BasePath 支持：所有静态引用带 `/projects/wx-chat/` 前缀
- Cache-Control：HTML 文档 `no-cache`，带版本令牌 `?v=0.0.1`
- healthz endpoint：返回 200 OK

### owner 需求来源

- e000001: 实现一个web微信聊天界面，只需要支持前端交互ui，不需要涉及接口设计和数据库

### 验收标准

- 页面在浏览器独立打开
- 联系人列表展示 3+ 联系人
- 点击联系人切换聊天窗口
- 输入消息并发送后出现在聊天区域
- 消息区分发送者和接收者气泡
- 桌面分辨率下布局正常
- healthz 返回 200
- `Cache-Control: no-cache` HTTP 响应头
- 静态资源 URL 带 `?v=0.0.1`

## 技术方案详设

### 组件树

```
App (flex row, 100vw x 100vh)
├── Sidebar (280px fixed width, #2E2E2E background)
│   ├── Header ("聊天" title)
│   └── ContactList (scrollable)
│       └── ContactItem[] (hover + active states)
│           ├── Avatar (CSS ::before color block, 40x40)
│           └── Name
└── ChatArea (flex: 1)
    ├── ChatHeader (current contact name)
    ├── MessageList (flex: 1, overflow-y: auto, scroll to bottom)
    │   └── MessageBubble[]
    │       ├── mine: green (#95EC69), float right
    │       └── other: white (#FFF), float left
    │       └── timestamp below bubble
    └── InputBar (fixed bottom, flex row)
        ├── Input (flex: 1, rounded)
        └── SendButton ("发送")
```

### 数据模型

```js
const VERSION = '0.0.1';
const BASE_PATH = '/projects/wx-chat/';

contacts: [
  { id, name, avatarColor, lastMessage }
]

messages: {
  [contactId]: [
    { id, from: 'me'|string, text, time: 'HH:MM' }
  ]
}
```

### 交互流程

1. 页面加载 → 渲染联系人列表 + 默认选中第一个联系人 → 渲染聊天记录
2. 点击联系人 → 切换 active 态 → 替换聊天区消息
3. 输入文字 + 回车/点击发送 → 追加到 messages[activeId] → 重新渲染 → 自动滚底
4. 新消息模拟"对方回复"不用实现（纯 UI 演示）

### 风险与缓解

| 风险 | 缓解 |
|------|------|
| 布局在非桌面分辨率下错乱 | 设置 min-width: 900px，滚动条兜底 |
| basePath 硬编码导致部署迁移困难 | 统一用 JS 变量 `BASE_PATH`，动态拼 URL |
| Cache-Control 遗漏导致 Hermes 验收拒 | server.py 对 `text/html` Content-Type 统一追加 `Cache-Control: no-cache` |
| 版本令牌遗漏 | index.html 内资源引用统一追加 `?v=${VERSION}` |
